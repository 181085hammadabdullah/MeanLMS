Educate-Me: (LMS website)
Developers Home

Educate-Me is an e-Learning LMS website, a complete full stack app, developed by Zaki-ul-Hassan and Hammad Abdullah as a final project for Full-Stack Web Development. The user interface of whole website is designed keeping in mind the the best user friendly graphics. All the front end of the website is designed in such a way that the website gives a professional look to the end user.

Website consists of three main modules.

  1. Client Interface
  2. Student LMS
  3. Admin Dashboard

A very remarkable feature of this website is, a Chat App for communication between User and admin. This is a realtime chat application for instant messages.

1 - Client Interface

Client Interface is the first module and it represents the main features of the website. It is designed for the general public. Any person visiting this website will see this module first after that he may navigate to the desired component.
Client Interface has following pages;
  • HOME
  • COURSES
  • NEWS & EVENTS
  • HOW IT WORKS
  • FAQS
  • CONTACT US
  • SIGN IN
  • SIGN UP

2 - Student LMS

After visiting Client Interface, User can Register and then Login to the Lms module. Lms is designed for students. Lms has all the posible features which any e-Learning website should have.
All the data posted on LMS is dynamic. Data comming from the 3rd module (ADMIN MODULE).
Student LMS has following features;
  • Student can Edit its profile
  • Student can update its Picture
  • Student can Enroll in any program
  • Student can view any News & Event Accouncement
  • Student can view any Assignment & Results
  • Student can Chat with the Admin

3 - Admin Dashboard

Admin Dashboard is designed to perform all the admin related tasks. It has all the posible features which an Admin Dashboard should have. Educate-Me Lms is a dynamic website. Admin can perform his tasks easily.
Admin Dashboard has following features;
  • Admin can Add & Edit Programs
  • Admin can Add Video Lectures
  • Admin can Add & Edit News & Events or Announcments
  • Admin can View all the students enrolled
  • Admin can Add Assignments & Results
  • Admin can Chat with Students
